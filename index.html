¡Bienvenidos a la guia básica de Git!

En el siguiente recorrido te mostraré los comandos base de Git que te serviran para crear, modificar y guardar tu propio repositorio ya sea localmente o en la nube.

git init

Este comando sirve para crear un nuevo repositorio de manera local.

git status

git status muestra los cambios que se han realizado en el directorio de trabajo. Por ejemplo haber añadido o modificado archivos. Cuando los archivos están en rojo no se han guardado los cambios, para eso esta el comando siguiente.

git add .

Se utiliza para agregar todos los cambios para el siguiente commit. Para agregar sólo un archivo se utiliza el nombre del fichero: git add index.html.

git log

Muestra todos los commits realizados anteriormente.

git commit

Se crea un commit con todos los cambios agregados en git add. Ejemplo: git commit -m "Mi primer commit"

git push

git push permite guardar todos los cambios de manera remota.

Crea un nuevo repositorio

Imagina que estas en una carpeta de proyecto con un archivo index.html. Crea un repositorio local, mira el estado del repositorio, añade index.html y vuelve a mirar el estado del repositorio.
